## [ Backend ]

### Backend 개발 진행 상황

#### User

- 일반 로그인
- 일반 회원가입
- 구글 로그인 => 테스트 필요
- 마이페이지 수정
- 회원 탈퇴

#### Recipe

- JSON 파일에서 레시피 데이터 읽어오기
- 레시피 데이터 보내기

#### Review

- 유저별 리뷰 작성하기
- 모든 리뷰 조회하기

#### Video

- 요리 영상 보기

#### Hashtag

- 레시피별 해시태그 정보 제공하기

#### Ingredient

- 선택된 재료에 따라 해당된 레시피 제공하기

#### Survey

- 유저의 취향을 반영할 수 있는 설문조사 진행하기 : 유저가 좋아하는 음식 / 유저가 가진 알레르기
